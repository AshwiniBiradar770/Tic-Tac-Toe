A simple, console-based Tic-Tac-Toe game implemented using core Java concepts. Perfect for practicing arrays, loops, conditionals, Scanner input, and OOP fundamentals. 
Features:
1. 3x3 game board using a 2D char array
2. Two players: X (first) and O (second)
3. Turn-based input via ScannerWin detection for rows, columns, and diagonals
4. Draw detection when board is full
5. Replay option for multiple games

Tech Stack:
Java 8+ (core features only: no external libraries)Console I/O with Scanner and System.out

How to Run
1. Clone or download this repository
2. Open terminal/command prompt in the project directory
3. Compile: javac TicTacToe.java
4. Run: java TicTacToe
5. Enter row (1-3) and column (1-3) numbers when prompted
6. Enjoy!

Game Rules:
1. Players alternate turns (X starts)
2. Enter positions as row,column (e.g., "1 2" for top-middle)
